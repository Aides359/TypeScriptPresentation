Was ist TypeScript?

  • Superset von JavaScript
  • Static Typing für JavaScript
  • Ideal für größere Projekte
  • Ideal beim Arbeiten mit APIs und Libraries

Warum TypeScript

  • Größere Projekte lassen sich leichter managen.
  • Compiletime exceptions wenn "falsche" Typenkonversionen gemacht werden.
  • Autocomplete in IDEs
  • Bereits jetzt tolle neue JavaScript features nutzen (ES6, ES7, ...)

ES-Was?

ECMAScript - Der Standard hinter JavaScript

umfasst neue Features für JavaScript

früher in Versionen (ES5, ES6, ES7), ab Version 6 in Jahreszahlen (ES2015, ES2016)

Interfaces

								{
									firstName: string;
									lastName: string;
								}

								function sayHello(person: Person): string
								{
									return "Hello " + person.firstName + " " + person.lastName;

Klassen


								class Student
								{
									public fullName: string;

									constructor(public firstName: string, public lastName: string)
									{
										this.fullName = firstName + " " + lastName;
									}

								interface Person 
								{
